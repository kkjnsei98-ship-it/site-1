Álgebra

A álgebra é o ramo da matemática que estuda a manipulação de equações, fórmulas e expressões matemáticas utilizando letras para representar números.

Expressões Algébricas

Uma expressão algébrica é uma combinação de letras (variáveis) e números conectados por operações matemáticas.

Exemplo: 3x + 2y - 5

Onde x e y são variáveis, 3 e 2 são coeficientes, e -5 é o termo constante.

Equações do 1º Grau

Uma equação do primeiro grau é uma igualdade que envolve uma ou mais variáveis elevadas ao expoente 1.

Exemplo: 2x + 5 = 13

Para resolver: 2x = 13 - 5 → 2x = 8 → x = 8/2 → x = 4

Sistemas de Equações

Um sistema de equações é um conjunto de duas ou mais equações com as mesmas variáveis.

Exemplo:

2x + y = 10

x - y = 2

Podemos resolver por substituição ou adição.

Geometria

A geometria estuda as formas, tamanhos, posições relativas e propriedades do espaço.

Ângulos

Ângulo é a região do plano limitada por duas semirretas de mesma origem.

Tipos de ângulos:

• Agudo: menor que 90°

• Reto: igual a 90°

• Obtuso: maior que 90° e menor que 180°

• Raso: igual a 180°

Teorema de Pitágoras

Em um triângulo retângulo, o quadrado da hipotenusa é igual à soma dos quadrados dos catetos.

a² = b² + c²

Exemplo: Se os catetos medem 3cm e 4cm, então a hipotenusa mede:

a² = 3² + 4² = 9 + 16 = 25 → a = √25 = 5cm

Áreas de Figuras Planas

Cada figura geométrica tem uma fórmula específica para calcular sua área.

Retângulo: A = base × altura

Triângulo: A = (base × altura) / 2

Círculo: A = π × raio²

Probabilidade

Probabilidade é o estudo das chances de ocorrência de um evento entre todos os resultados possíveis.

Conceitos Básicos

Espaço Amostral (S): Conjunto de todos os resultados possíveis de um experimento.

Evento (E): Subconjunto do espaço amostral.

Cálculo de Probabilidade

A probabilidade de um evento E ocorrer é dada por:

P(E) = número de casos favoráveis / número de casos possíveis

Exemplo: Qual a probabilidade de sair um número par ao lançar um dado?

Casos favoráveis: {2, 4, 6} → 3 casos

Casos possíveis: {1, 2, 3, 4, 5, 6} → 6 casos

P(par) = 3/6 = 1/2 = 0,5 = 50%

Probabilidade de Eventos Compostos

Para eventos independentes: P(A e B) = P(A) × P(B)

Para eventos mutuamente exclusivos: P(A ou B) = P(A) + P(B)

Funções

Função é uma relação entre dois conjuntos, onde cada elemento do primeiro conjunto (domínio) está associado a um único elemento do segundo conjunto (contradomínio).

Conceito de Função

Uma função f: A → B associa a cada elemento x do conjunto A um único elemento y do conjunto B.

Exemplo: f(x) = 2x + 1

Para x = 3, temos f(3) = 2×3 + 1 = 7

Tipos de Funções

Função Afim: f(x) = ax + b (gráfico é uma reta)

Função Quadrática: f(x) = ax² + bx + c (gráfico é uma parábola)

Gráfico de Funções

O gráfico de uma função representa visualmente a relação entre x e f(x).

Exemplo: Para f(x) = x + 2

Quando x = 0, f(x) = 2 → ponto (0,2)

Quando x = 1, f(x) = 3 → ponto (1,3)

O gráfico é uma reta que passa por esses pontos.
